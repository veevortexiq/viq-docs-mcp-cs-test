    • Additional addresses

    • Shipping lines

    • Tax lines

    • Discount allocations

    • Transactions

    • Fulfillments

    • Tracking numbers

    • Refunds

    • Tags

    • Note attributes

    • Order metafields

    • Minimum purchase

    • Eligibility rules

    • Start/end dates

    • Blog title

    • Blog handle

    • Author

    • Tags

    • SEO fields

    • Handle

    • HTML content

    • Template

    • Page metafields

    • File media (images, videos, PDFs)

    • File URL

    • Alt text

    • Metadata

    • Binary ingestion support

    • Liquid templates

    • Sections

    • Snippets

    • Layouts

    • CSS/JS assets

    • Theme settings (settings_data.json)
