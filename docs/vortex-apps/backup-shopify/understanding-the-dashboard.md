# Understanding the Dashboard

![Dashboard1.png](../../img/shopify-backup/Dashboard1.png)

The dashboard displays the following details:

- Store Name

- Store Identifier

- Date of Last Store Data Backup

- Timestamp of the Most Recent Change to Any Data Entity

- Data Sync Status (Active or Inactive)

- Catalogue Information:

- Total count of Products and Collections

- Realtime backup status (enabled or not)

- Time of the last realtime backup (sync)

- Sync status

- Action: Clicking the View button redirects to the [Restore Centre](understanding-restore-center.md) page.
