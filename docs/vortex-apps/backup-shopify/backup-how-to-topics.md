# How to Topics

**A) Enabling/disabling realtime backup**

Settings &gt; Click on the toggle button under the status column corresponding to the data entity

Click on the Change Logs tab, to view the event log entry is success

**B) Manual backup of data entities not having realtime webhook updates**

The data entities identified as Manual in the column Backup Type, click on the corresponding Backup button in the Action column.

Then click Confirm button displayed in the subsequent popup window.

Click on the Change Logs tab, to view the event log entry is success.
