# StagingPro Setup

Install the app from the BigCommerce App Marketplace at the following url:

<a href="https://www.bigcommerce.com/apps/stagingpro/" title="StagingPro App" target="_blank" rel="noopener">https://www.bigcommerce.com/apps/stagingpro/</a>

The pricing page will be displayed with the subscription plan.

Click on the subscription button to make the payment and then you will be ready to use the app.

As soon as the payment is done, the app license is activated.
