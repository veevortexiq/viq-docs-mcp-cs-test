# Understanding the History and Rollback Page

The History and Rollback page provides a detailed view of your past and scheduled activities. It includes the following tabs:

### Data Migrations

View all historical migration records, including:

- Request ID
- Source Environment
- Destination Environment
- Start Date & Time
- Migration Status (with a View Details option)
- Completion Date

### Code Deployment

- Visible when GitHub integration is enabled.
- This section displays deployment activities triggered by branch merges and workflow processes.
- Refer to the GitHub Actions / Workflow Process documentation for more details.

### Schedule Migration

View your upcoming scheduled migrations.
This feature is available in **Premium** plan only.
