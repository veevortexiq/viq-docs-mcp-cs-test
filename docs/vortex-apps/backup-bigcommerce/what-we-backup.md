# Data Entities for Backup and Restore

## List of Data Entities and corresponding fields that we Backup and Restore


### 1. Products

* Title

* Description / bodyHtml

* Handle

* Product type

* Vendor

* Tags

* Status

* Options (Option1 / Option2 / Option3)

* Variants (Title, SKU, Barcode, Price, Compare-at price, Inventory policy, Weight, Fulfillment service)

* Images (Src, Alt text, Position)

* SEO fields (title, description)

* Metafields

* Collections linkage

* Media attachments

### 2. Collections

* Title

* Description

* Handle

* Manual & automated collection types

* Rule conditions

* Sort order

* Image (src, alt text)

* SEO fields

* Collection metafields

### 3. Customers

* First name

* Last name

* Email

* Phone

* Tags

* Note

* Marketing preferences

* Tax exemption

* Default address

* Additional addresses

* Customer metafields

### 4. Orders

* Order number

* Email

* Billing and shipping addresses

* Line items (product, variant, quantity, price)

* Shipping lines

* Tax lines

* Discount allocations

* Transactions

* Fulfillments

* Tracking numbers

* Refunds

* Tags

* Note attributes

* Order metafields

### 5. Discount Codes

* Code

* Title

* Percentage or fixed value

* Applies-to rules

* Usage limits

* Customer eligibility

* Minimum purchase

* Start/end dates

* Discount metafields

### 6. Automatic Discounts

* Discount type (BOGO, percentage, fixed)

* Conditions

* Eligibility rules

* Start/end dates

### 7. Blogs

* Blog title

* Blog handle

* Blog metafields

### 8. Blog Posts

* Post title

* Author

* HTML content

* Featured image and alt text

* Tags

* SEO fields

* Post metafields

### 9. Pages

* Title

* Handle

* HTML content

* Template

* SEO title & description

* Page metafields

### 10. Files (Images)

* File media (images, videos, PDFs)

* File URL

* Alt text

* Metadata

* Binary ingestion support

### 11. Themes

* Liquid templates

* Sections

* Snippets

* Layouts

* CSS/JS assets

* Theme settings (settings\_data.json)

* Theme metafields

### 12. Gift Cards

* Code

* Balance

* Expiry date

* Associated customer

* Status

* Gift card metafields

### 13. Metafields (All Entities)

* Products

* Variants

* Collections

* Customers

* Orders

* Pages

* Blogs

* Theme metafields

* File metafields

* Gift card metafields
