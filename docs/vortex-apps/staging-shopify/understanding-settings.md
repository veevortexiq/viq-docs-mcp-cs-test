# Vortex Staging Setup

The Settings screen is where you configure how you want to be notified when environment statuses change or when migration activity completes.

The main area used for this is the Status Notifications tab.

![US1.png](../../img/shopify-staging/US1.png)

Vortex Staging supports the following notification methods:

> [Email](staging-setup-email.md)

> [Microsoft Teams](staging-setup-teams.md)

> [Slack](staging-setup-slack.md)
