Welcome to the Vortex Staging Wiki, where you will find many articles on how best to use Staging app or to undertake any required troubleshooting.

Staging simplifies staging with automated workflows, real-time migration monitoring, and secure data handling. 

Its seamless integration with GitHub and other AI agents ensures efficient deployments and team collaboration. With features like selective migration, scheduled rollouts, and rollback support, it optimises operations while reducing errors and saving valuable time.

Whether you are a store owner or an agency with dozens of team members, Staging has been designed to get you up and running within minutes. 

Refer to [Staging Onboarding](staging-onboarding/index.md) as reference to quickly get started.
