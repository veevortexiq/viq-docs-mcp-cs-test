# Email Sender - Dot Digital Agent

## Agent Overview
This agent helps you send emails through your DotDigital account. It will ask for the necessary details like the recipient's email address, the subject, and the message content. Once it has all the required information, it will send the email for you.

## Agent Tools
- **Send Email**: This is the primary tool used by the agent. It takes the recipient's email address, the subject line, and the message you provide to send an email using your connected DotDigital account.

## Sub-agents

This agent does not have any sub-agents.

## Scenarios
Here are a few scenarios where this agent can be particularly useful:

### Scenario 1: Sending a Welcome Email to a New Customer
You can use this agent to quickly send a personalized welcome email. Just provide the new customer's email address, a welcoming subject, and the message you want to send.

### Scenario 2: Distributing a Promotional Offer
If you need to send a special offer to a list of subscribers, you can provide the agent with the list of email addresses, the promotional subject, and the offer details. The agent will then handle sending the email to all recipients.

### Scenario 3: Quick Internal Team Updates
For sending a quick update to your team members, you can use the agent to send a plain text email. Simply provide the internal email addresses, a subject for the update, and the message you need to convey.
