# Page Speed and Web Vitals Agent

## Agent Overview

The PageSpeed and Web Vitals Agent is your personal guide to understanding and enhancing your website's performance and search engine optimization (SEO). Think of it as a helpful assistant that translates complex website data into simple, easy-to-understand advice. The agent's primary role is to check your website's speed and overall user experience, then offer to connect you with specialized agents who can make targeted improvements. It is designed to speak in plain language, avoiding confusing acronyms and technical terms, so you can clearly understand how well your site is performing.

## Agent Tools

This agent is equipped with tools to gather insights about your website:

*   **Today's Performance Check:** This tool provides an immediate snapshot of your website's current performance. It examines key metrics for both desktop and mobile users, giving you a real-time view of your site's speed and how quickly it responds to visitors.
*   **Historical Performance Review:** This tool looks at your website's performance history over the past few months. It helps you understand trends in your site's user experience, making it easy to spot any ongoing issues that might need attention.

## Sub-agents

The agent collaborates with a specialist to help you refine your website's content:

*   **VortexIQ Page & Widget Editor:** This sub-agent is an expert in editing the content within your website's widgets. It can assist you in making precise changes to improve your site's SEO and ensure your brand's message is clear, consistent, and impactful.

## Scenarios

Here are a few ways this agent can help you:

### Scenario 1: Getting a Quick Website Health Check
*   **You ask:** "What is my site's overall Performance Score for today?"
*   **The Agent does:** The agent uses its "Today's Performance Check" tool to fetch the latest performance scores for both the desktop and mobile versions of your website. It then explains the scores in simple terms—letting you know if they are "Good," "Need Improvement," or "Poor"—and offers a straightforward, non-technical tip to help you improve them.

### Scenario 2: Understanding Your Website's Past Performance
*   **You ask:** "How quickly did my site's main content load for mobile visitors over the last 90 days?"
*   **The Agent does:** The agent activates its "Historical Performance Review" tool to analyze loading speed data from the past three months. It will then tell you the average time it took for your main content to load for mobile users and explain whether that speed is ideal or could be improved, suggesting simple fixes like optimizing your images.

### Scenario 3: Improving Your On-Page Content for Better SEO
*   **You ask:** "My SEO score seems a bit low. Can you help me with that?"
*   **The Agent does:** After reviewing your site's performance, the agent recognizes that enhancing your on-page content could give your SEO a boost. It will offer, "Now that we've looked at your site's performance, would you like to edit some of the on-page content, like the text in your widgets?" If you agree, it will seamlessly connect you with its specialist sub-agent, the **VortexIQ Page & Widget Editor**, to guide you through making those improvements.
